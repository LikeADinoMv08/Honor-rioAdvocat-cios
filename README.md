# Honorários Advocatícios (OAB 2025)

Sistema Jurídico para cálculo de honorários advocatícios e geração de propostas, com base na Tabela OAB 2025 (SP e PR disponíveis no momento).

## Tecnologias Usadas

- React (Vite)
- Tailwind CSS
- Lucide React

## Como Executar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
